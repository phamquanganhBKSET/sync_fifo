### Directory layout
.
├───hdl         # HDL code                                       <br />
├───inc         # Include file which declares all constants      <br />
├───libs                                                         <br />
│   └───model   # Model which is used to check outputs of design <br />
└───sim         # Simulation files                               <br />
    ├───tb      # Top testbench                                  <br />
    └───work    # Script for simulation                          <br />

### How to simulate

2. Run the following commands
  +) # source run_bash.sh

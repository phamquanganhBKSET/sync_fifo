### Directory layout
.
├───hdl         # HDL code
├───inc         # Include file which declares all constants
├───libs
│   └───model   # Model which is used to check outputs of design
└───sim         # Simulation files
    ├───tb      # Top testbench
    └───work    # Script for simulation

### How to simulate
1. Go to directory sim/work
2. Run the following commands
  +) # source run_bash.sh
  +) # vc
